# 📊 Sales Performance Dashboard using Power BI

An interactive **Sales Performance Dashboard** built using **Power BI** to analyze business performance and provide actionable insights. This project demonstrates data visualization, KPI tracking, DAX calculations, and interactive reporting for business decision-making.

---

## 📌 Project Overview

The objective of this project is to design a professional and interactive dashboard that enables stakeholders to monitor sales performance, profitability, customer segments, and regional trends. The dashboard allows users to filter data dynamically and gain valuable business insights.

---

## 🎯 Objectives

- Analyze sales and profit performance.
- Track key business KPIs.
- Monitor monthly sales trends.
- Compare sales across categories and regions.
- Identify top-performing products.
- Build an interactive dashboard using Power BI.

---

## 📂 Dataset

**Dataset:** Sample Superstore Dataset

The dataset contains sales transactions with information such as:

- Order Date
- Sales
- Profit
- Quantity
- Discount
- Category
- Sub-Category
- Product Name
- Customer Segment
- Region
- Ship Mode

## 📈 Dashboard Features

- 📌 KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders
  - Profit Margin

- 📊 Monthly Sales Trend Analysis

- 📦 Sales by Category

- 🌍 Profit by Region

- 👥 Sales by Customer Segment

- 🏆 Top 10 Products by Sales

- 🎛 Interactive Slicers
  - Region
  - Category
  - Segment
  - Ship Mode
  - Year

---

## 📊 Key Insights

- Technology is one of the highest-performing product categories.
- Sales and profits vary significantly across different regions.
- Consumer segment contributes a major share of total sales.
- Monthly trend analysis helps identify seasonal sales patterns.
- Interactive filters enable quick business analysis from multiple perspectives.

3. Explore the interactive dashboard.
4. Use slicers to filter data by Region, Category, Segment, Ship Mode, and Year.
